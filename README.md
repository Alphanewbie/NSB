## 수화 언어란?

수화언어는 **제한된 시간 내**에 **한정된 표현**을 통해 원하는 의미만을 전달해야 하므로, 허사(虛辭; 문법적 기능을 나타내는 조사, 어미 등)를 최대한 생략하고 **실사(實辭) 위주**로 발달했다.
